# Maker Workshop Avatar Icon Pack

Transparent individual PNG avatar/icon pack for Jinn/Cuttlefish.

## Contents

- 50 individual icons
- Transparent backgrounds
- Sizes: 64, 128, 256, 512
- Flat icon folders under `icons/<size>/`
- Jinn-compatible paths under `packages/web/public/avatars/maker-workshop/<size>/`
- `manifest.json`
- `icon_descriptions.json`
- `packages/web/src/lib/maker-workshop-avatar-pool.ts`
- source sheets and preview sheet

## Categories included

- 3D printers
- CNC machines
- Laser engravers
- Welders
- Rotary tool
- Filament spools
- Sewing thread spools
- Sewing machines and sewing needs
- Soldering, measuring, and maker tools
- Personal protective equipment
