# Garage Workshop Avatar Icon Pack

Transparent individual PNG avatar/icon pack for Jinn/Cuttlefish.

## Contents

- 70 individual icons
- Transparent backgrounds
- Sizes: 64, 128, 256, 512
- Flat icon folders under `icons/<size>/`
- Jinn-compatible paths under `packages/web/public/avatars/garage-workshop/<size>/`
- `manifest.json`
- `icon_descriptions.json`
- `packages/web/src/lib/garage-workshop-avatar-pool.ts`
- source sheets and preview sheet
