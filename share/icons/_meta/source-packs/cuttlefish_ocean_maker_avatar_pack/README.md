# Cuttlefish Ocean-Maker Avatar Pack

Second avatar pack for the Jinn/Cuttlefish repo format.

## Summary

- **Count:** 40 icons
- **Theme:** same glossy cartoon/sticker style as the first ocean pack, extended with maker, workshop, archaeology, optics, and tech icons
- **Sizes:** 64, 128, 256, 512
- **Asset root:** `packages/web/public/avatars/ocean-maker`
- **Served root:** `/avatars/ocean-maker`

## Directory layout

```text
packages/web/public/avatars/ocean-maker/64/*.png
packages/web/public/avatars/ocean-maker/128/*.png
packages/web/public/avatars/ocean-maker/256/*.png
packages/web/public/avatars/ocean-maker/512/*.png
packages/web/src/lib/ocean-maker-avatar-pool.ts
manifest.json
source-sheets/
docs/ocean-maker-avatar-preview.png
```

## Included icon groups

- Benchy / 3D printers / Geppetto / Pinocchio
- Archaeology tools
- CNC / optics / microscopes / binoculars / telescope / astrolabe
- Pirate flag / compasses / satellite / laptop / phones / walkie-talkie
- Woodworking hand, power, and standing tools

## Notes

- These were generated as 4 sticker sheets, then cropped into individual square PNGs.
- The icons preserve the same cute polished avatar style used for the first ocean pack.
