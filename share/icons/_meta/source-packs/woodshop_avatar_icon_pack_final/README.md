# Woodshop Avatar Icon Pack

Transparent individual PNG avatar/icon pack for Jinn/Cuttlefish.

## Contents

- 40 individual icons
- Transparent backgrounds
- Sizes: 64, 128, 256, 512
- Flat icon folders under `icons/<size>/`
- Jinn-compatible paths under `packages/web/public/avatars/woodshop-hand-tools/<size>/`
- `manifest.json`
- `icon_descriptions.json`
- `packages/web/src/lib/woodshop-avatar-pool.ts`
- source sheets and preview sheet

## Added tool classes

Planes, saws, chisels, mallets, boring tools, marking/layout tools, hammers, screwdrivers, nails, screws, and old-style clamps.
