# Woodworking Power Tool Icon Pack

Transparent cartoon-style icon/avatar pack inspired by the uploaded reference set.

## Contents

- 40 individual tool/color variants
- Categories: drills, drivers, grinders, batteries, vacuums, saws, routers, sanders
- Sizes: 64, 128, 256, 512
- Transparent PNGs only
- Generic-looking tools with no logos or brand names
- `manifest.json` and `icon_descriptions.json` included
- Preview sheet included under `docs/`

## Tool Types

- Cordless Drill
- Impact Driver
- Angle Grinder
- Cordless Battery Pack
- Shop Vacuum
- Jigsaw
- Circular Saw
- Trim Router
- Orbital Sander
- Reciprocating Saw

## Colorways

- Green
- Red
- Yellow
- Orange

## Folder Layout

- `icons/` contains the master 512px PNG for each asset
- `64/`, `128/`, `256/`, `512/` contain size-specific exports
- `docs/` contains a transparent-checker preview sheet
- `manifest.json` includes file paths and metadata
- `icon_descriptions.json` provides a simpler file-to-description mapping
