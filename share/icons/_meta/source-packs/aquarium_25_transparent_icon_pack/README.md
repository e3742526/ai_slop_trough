# Aquarium 25 Transparent Icon Pack

Jinn/Cuttlefish-compatible aquarium icon subset remade with individual transparent-background PNG files.

## Contents

- 25 individual transparent PNG icons
- Core groups: 5 tanks, 5 aquatic plants, 5 shrimp, 5 crabs, 5 turtles
- Sizes: 64, 128, 256, 512
- Individual 512px PNGs under `icons/`
- Asset root: `packages/web/public/avatars/aquarium-25-transparent`
- Served root: `/avatars/aquarium-25-transparent`
- Includes `manifest.json`, `icon_descriptions.json`, TypeScript pool, preview sheet, and source sheets.

## JSON

- `manifest.json` includes repo path metadata and descriptions.
- `icon_descriptions.json` is a simpler icon-description map for agent/UI use.
