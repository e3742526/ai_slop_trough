# Other People's Slop

Curated links to external generated-asset collections and icon-generation tools that are worth indexing without copying into this repo.

## Categories

At the moment this page has one category: `Icons`.

## Icons

| Name | Repo | Type | Description |
| --- | --- | --- | --- |
| Oddicons `GPL-3.0` | [jasperdevs/oddicons](https://github.com/jasperdevs/oddicons) | Asset pack | Free, weird, AI-generated icons. The repo points users to `oddicons.net` to browse, copy, or download icon sets and routes new icon requests through GitHub issues rather than direct asset submissions. |
| AI Icons Collection `MIT` | [espiria/icons](https://github.com/espiria/icons) | Asset pack | Various AI-made icons for apps and programs, mostly generated with OpenAI image models. The repo organizes assets into app-specific folders, including Kitty and Wezterm variants, and notes ongoing `.icns` conversion work for the original 1024×1024 images. |
| Lobe Icons `MIT` | [lobehub/lobe-icons](https://github.com/lobehub/lobe-icons) | Asset pack | Popular AI / LLM model brand SVG logo and icon collection with an online gallery plus static SVG, PNG, WebP, and package-based distribution options. Useful when you need recognizable model or company marks rather than generated illustrations. |
| Vectra Icons `MIT` | [abhishekshankr/vectra-icons](https://github.com/abhishekshankr/vectra-icons) | Asset pack | A 32×32-grid icon set for generative-AI applications with outlined and filled styles. Not AI-generated itself, but designed to represent AI concepts consistently across products, with a companion Figma plugin. |
| AI Icon Generator `MIT` | [samzong/ai-icon-generator](https://github.com/samzong/ai-icon-generator) | Tool | An OpenAI `gpt-image-1`-based icon generation tool focused on prompt-driven icon creation, preset styles, batch workflows, and export to PNG, ICO, ICNS, or JPEG. Useful when you want generation capacity instead of a fixed pack. |
| icon-gen-ai `license unclear` | [yauheniya-ai/icon-gen-ai](https://github.com/yauheniya-ai/icon-gen-ai) | Tool | A CLI and Python package for AI-assisted icon generation and transformation, with color, size, background, and export controls for SVG, PNG, WebP, and ICO. Better thought of as reproducible tooling than a ready-made icon dump. |
